Bass synth with frequency ramp. Create swoopy or sharp frequency ramp downs. Then run it through built-in effects and sequencer.

## Details

2 page combo patch: control the Rampi Bass synth, and choose an effect.  The Aux button controls a sequencer.

This is a multipage patch - each knob will have different roles depending on page.
