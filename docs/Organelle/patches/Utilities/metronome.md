A Link-enabled metronome patch with accent control and keyboard synth for tuning up the band.

## Details

A basic metronome patch.  Start and stop the metronome with the Aux button.  Control tempo and select a beat accent (or none).  The keyboard plays a super simple synthesizer tone that is useful for tuning up.

Link enabled for syncing with other devices on a shared wireless network.

Knob1: Tempo

Knob2: Accent Placement

Knob3: Metronome Tone

Knob4: Synth Tone

Aux: Metronome Start/Stop
