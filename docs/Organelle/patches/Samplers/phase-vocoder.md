A classic phase vocoding sampler. Recorded samples loop with independent playback speed and pitch! Stretch and trim a complicated sound into a rhythmic loop or compress it into a single note.

## Details

Record a sound up to 30 seconds in length and then loop it. Set loop start point and loop length, control speed, direction, and pitch.The keys play the sample at different pitches. The phase vocoder allows you to change pitch and speed independently so a sound can be slowed down to zero while preserving the pitch.  This leads to some very strange sounds.  It is also useful as a practice tool for transcribing fast lines.

Knob1: Playback Speed

Knob2: Tuning

Knob3: Sample Start Point

Knob4: Sample Length

Aux and Foot Switch: Hold to Record

To save your sample, use the 'Save' or 'Save New' command in Storage menu.
