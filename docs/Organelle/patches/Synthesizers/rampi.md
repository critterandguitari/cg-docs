Ramping swoop-y synthesizer with sequencer and waveform select.

## Details

Knob1: Ramp Attack

Knob2: Ramp Decay

Knob3: Ramp Depth

Knob4: Release Time

Aux: Sequencer Control + Waveform Control. Use top row of keys to select options in the Menu. Lower row of keys selects the sequence record 'slot.'
