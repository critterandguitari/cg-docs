## Details

Before we get into the specifics of this patch, please know that this patch is provided here in the '.zop' (Zipped Organelle Package) format. This ensures proper installation on your Organelle. There are differences in processing power between the original and 'M' versions of the Organelle. The .zop installation will check which version of Organelle before installing the appropriate number of effects. If you have an original Organelle, Cannabits will use one effect and lack a 'Aux Menu'. The Organelle M & S .zop installation will install a version of Cannabits where up to three effects can be used simulateneously. The Organelle M & S version also includes an Aux Menu.

To install the .zop file, simply upload the file to your SD card or copy to your USB Drive as you normally would with other patch files. After selecting 'Reload' from the Storage menu, use the Organelle's encoder to navigate to the file location and select 'Install Cannabits.zop.' The OLED will then display some installation information and exit upon completion.

OK...back to the patch:

Cannabits selects a new effect at a rate of your choosing. You don't get to pick the effect or set the parameters of the effect, so it's pretty 'hands off' once you set the rate and the wet/dry mix.

There are 12 effects the patch chooses from:   
*bit crusher, chorus, comb filter, constant gardener (playback speed and pitch effect), delay, distortion, envelope follower, freeze, pitch shifter, reverb, ring modulator, and sines (aka 'partial party' port)*.

Once the patch picks an effect, it also randomly picks the parameters for the effect.

**How do I use this patch?**

A typical use case would be:

1. Connect a sound to the line input jack.  
2. Set rate at which random effects are chosen.    
3. Set Dry/Wet audio mix.  
4. Use the keys to revisit previous effects.  
5. In the case of the Organelle M or S, adjust the number of effects chosen at once.  
6. Revisit the above steps in any order as needed.

*For Organelle M/S:*

To do some of the steps above you will need to use the ‘Aux Menu’. The Aux Button controls the ‘Aux Menu’. When you press Aux for 0.2 seconds, a new menu is displayed on OLED Screen for as long as you hold Aux. To select an Aux Menu command/Page, use the Top Row of Organelle keys. There are 10 possible commands selected by the 10 keys. The Low C# key selects Top Left command. The High A# key selects the Bottom Right command. The Command Grid is to be read from top left to bottom, then top right to bottom.

The following commands are available on the Aux Menu's Left Column:

Play/Pause - Plays or Stops randomization.  
1Fx: On/Off - If 'On', the patch will randomly generate one effect.   
2Fx: On/Off - If 'On', the patch will randomly generate two effects.   
3Fx: On/Off - If 'On', the patch will randomly generate three effects.

_________  
*For Organelle 1:*   
There is no Aux Menu. The Aux Button plays or stops randomization of one effect.  
________

Knob Parameters are identical on the Organelle M/S and 1:  
Knob1: Sets the tempo.  
Knob2: Selects rhythmic note type.  
Knob3: Sets the quantity of rhythmic notes.   
Knob4: Sets the Dry/Wet mix. 0% is no effect heard; 100% is only effected sound heard.

The Organelle's keyboard selects the last twenty-four (24) effects. The newest is the 'C' next to the Aux button. The oldest is the high 'B' key.

If you like a setting, don’t forget to ‘Save’ in the Storage menu.

**Tech Specs:**

Sync tempo with MIDI clock or Ableton Link!

Footswitch: Starts/Stops the randomization.

**Fun Thing:**

Send Cannabits both a MIDI clock and audio from a drum machine - the audio and effects will be in sync!
