Freeze by Tempo is a ‘freezer’ effect that is synced to tempo. A short audio sample is frozen (aka seamlessly looped) until the next user-set tempo event occurs.

## Details

Note: Installation on Organelle 1 (original) vs. Organelle M / S: This patch is released in the .zop format to account for CPU differences between the two Organelle models.

This patch uses two patch ‘pages,’ keyboard controls, and the ‘Aux Menu’ to control recording, playback, and mix down. The keyboard controls can be used on either page.

A typical use case would be:  
1. Send audio to the Organelle’s Audio Input and MIDI Clock to the TRS-MIDI input port or USB MIDI.  
2. Set the freeze interval by selecting note type and amount  
3. Set the length of the freeze.  
4. Adjust Wet/Dry mix.  
5. Use keyboard to transpose frozen sample.  
6. Revisit any/all of steps 2-5 to suit your sound

To do most of the steps above you will need to use the ‘Aux Menu’. The Aux Button controls the ‘Aux Menu’. When you press Aux for 0.2 seconds, a new menu is displayed on OLED Screen for as long as you hold Aux. To select an Aux Menu command/Page, use the Top Row of Organelle keys. There are 10 possible commands selected by the 10 keys. The low C# key selects Top Left command. The High A# key selects the Bottom Right command. The Command Grid is to be read from top left to bottom, then top right to bottom.

The following command is available on the Aux Menu’s Left Column:

Playing/Stopped: Toggles the freeze on/off.

The following commands are available on the Aux Menu’s Right Column:

Main On/Off: Turns the main page on/off. See below for more information about this page.  
BPM On/Off: Turns the tempo page on/off. See below for more information about this page.

The Bottom Row of keys do not apply to the Aux Menu of this patch.

Main Page parameters:  
Knobs 1&2 set freeze interval.  
Knob1: Note Type: 16th – Whole notes  
Knob2: Note Amount: 1x – 16x  
Knob3: Sets Length of freeze. Maximum length is the freeze interval set by Knobs1&2.  
Knob4: Wet/Dry mix.  
Line 5: Indicates current tempo and transposition.

BPM Page parameters:  
Knob1: Sets tempo if no tempo is found via Link or MIDI clock.  
Knob2: Fine tuning of the tempo  
Knob3: n/a  
Knob4: n/a  
Line 5: Indicates current tempo and transposition.  
______

TECH SPECS:  
Link enabled.

Foot Switch: Starts / Stops freeze effect.
