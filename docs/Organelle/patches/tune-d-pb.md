Tune D PB is a multipage synth built around percussive samples. Delay synthesis, effects and a sequencer, all in one patch!

## Details

This 3-page patch combines Tuned Delay and sequencing from Pow Pow's Polybeats with an array of audio effects. Control a tuned drum feedback synthesizer, choose an effect, and key in a poly beat sequence. Knob parameters are specific to each page.

Aux and Footswitch: (short press) Toggle On/Off Sequence Latch; (long press) Clear Latched Notes

This patch Link enabled for syncing with other devices on a shared wireless network.
