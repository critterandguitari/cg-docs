# Update-OS-v2.1


**Tags:** OS Update, Utility

**[Download Patch](http://thepeacetreaty.org/organelle/patches/Update-OS-v2.1.zip)**

This patch updates the Organelle OS to v2.1. Please see instructions below!

## Details

This patch updates the Organelle OS to v2.1. *Run this update **ONLY** if you have an OS that is older than 2.1!* Before downloading this patch, check your current version by navigating to Info under the System menu.

To update:

1. Download and Unzip the update patch. Do NOT change the name of folder! If you do, the update will not work.

2. Eject the USB drive from Organelle and insert in your computer.

3. Copy unzipped folder to the Patches folder on USB drive, eject, and reinsert in Organelle. Select Reload from the System menu.

4. Navigate to and select the update patch from Patch menu. Follow instructions on OLED screen (You will be prompted to press Aux button to start update). Please be patient and do not interrupt process!

5. The Organelle shuts down after the update. You will have re-apply power.
