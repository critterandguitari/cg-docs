## Details

Wepa! plays back a sample, pitched based on which key you play. Holding a key loops sample playback. There's delay too!

Change the .wav file in the patch folder to play your own sample.

Knob1: Playback Speed

Knob2: Loop Length

Knob3: Delay Time

Knob4: Delay Feedback

Aux and Foot Switch: Playback Speed Shift
