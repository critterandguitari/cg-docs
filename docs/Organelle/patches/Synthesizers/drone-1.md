Harmonic drone synthesizer with sequencer.

## Details

Each key is an overtone on the harmonic series.  You can adjust the volume rise and fall time from < 1 second to 60 seconds making it easy to control a slowly evolving tone.

Knob1: Base Frequency

Knob2: Detune

Knob3: Rise/Fall Time

Knob4: Brightness

Aux: Sequencer
