Three page combo patch: adjust the sampler, choose an effect, and use polybeat sequencer to control sample playback.

## Details

This is a multipage patch - each knob will have different roles depending on page.

Aux: Starts/Stops Polybeat sequencer

Foot switch: Controls recording of new sample - two seconds max record time.

To save your sample, use the 'Save' or 'Save New' command in Storage menu.

Link enabled for syncing with other devices on a shared wireless network.
