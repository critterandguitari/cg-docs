## Details

Drummy lets you prepare a sound for each key. Tune up a different drum sound on each key by customizing start frequency, ramp time, noise and decay and then record a sequence. This patch has two states: play state and edit state.

Play State is the default. Parameters:

Knob1: HFO Frequency

Knob2: HFO Depth

Knob3: Delay Time

Knob4: Delay Feedback

Aux: Sequencer Control. Use top row of keys to select options in the Menu. Lower row of keys selects the sequence record 'slot.'

Foot Switch: N/A

To enter Edit state: Press and hold Aux key to enter Aux menu and press high C#. The LED will become white. Press key to start editing. Edit state parameters:

Knob1: Starting Frequency

Knob2: Ramp Down Duration

Knob3: Noise

Knob4: Decay Time

Foot Switch: N/A

To exit Edit State: Press and hold Aux key to enter Aux menu and press high C#.

Pro Tip: In edit state: Copy parameters from one key to another by pressing and holding a key and then press a second 'destination' key. This is useful for creating many similar sounds.

Don't forget to save your work!  Exit Edit State and then use the 'Save' or 'Save New' command in Storage menu.
