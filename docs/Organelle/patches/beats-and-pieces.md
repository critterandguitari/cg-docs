## Details

Play a song on the fly! Live remix it!

This patch shines when you load your samples into it (44.1KHz / 16-bit / .wav) The Aux button toggles a background/drum track (should be called 'drums.wav'). Each keyboard key controls a sample (1.wav - 24.wav). The knobs allow control of the background track volume and sample playback decay and a delay line for extra fun.  The background track is streamed from disk so it can be as long as desired.  The key samples get loaded when the patch begins, so they should be shorter (under a minute).

Knob1: Drum Track Volume

Knob2: Sample Release Time

Knob3: Delay Time

Knob4: Delay Feedback

Aux: Toggles Drum Track Playback

Foot Switch: N/A
