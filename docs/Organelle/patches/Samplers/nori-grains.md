**[Download Patch](https://patchstorage.com/wp-content/uploads/2018/11/Nori-Grains.zip)**

Record a sound and create a randomized granular synth cloud from it. Keys transpose the cloud. Set sample amplitude rise-fall times from 1-60 seconds. Great for creating drones and strange percussion.

## Details

Press and hold the Aux button to record. Each key will activate grain voice that fades in and out. The control the fade in / out time to create slowly evolving grain clouds. Play up to 16 keys at once. The detune control effects the output on the Left output only, creating a slight stereo dissonance. The OLED Keyboard shows which keys have been pressed and their amplitude state (key gets more filled in as amplitude rises, less as it falls)

Knob1: Grain Size

Knob2: Grain Density

Knob3: Detuning

Knob4: Rise & Fall Time

Aux and Foot Switch: Press to Record New Sample

To save your sample, use the 'Save' or 'Save New' command in Storage menu.
