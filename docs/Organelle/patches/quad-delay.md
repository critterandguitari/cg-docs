## Details

Audio input is fed through four delay lines, each with a delay time that is a multiple of a root delay time.  The root delay time selectable with the keyboard (Low 'C' key = 0ms delay. High 'B' key = 918 ms delay). Knobs control feedback. One knob for each delay line.
