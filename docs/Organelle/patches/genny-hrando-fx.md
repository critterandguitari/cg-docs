## Details

Three page generative, sample-playing patch: randomizing arpeggio, sampler, effect chooser.  Because this is a multipage patch each knob will have different roles depending on page.

Thanks to Pow Pow for the sounds. You can replace the samples used in this patch with your own. Each numbered folder in the HRando folder can have up to four samples (44.1KHz / 16-bit / .wav).
